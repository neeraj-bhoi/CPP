/*
    if you take pointer of class B in class A
    we say that class A is composed with class B as a member

    if you take reference of class B in class a
    we say that class A is aggregated with class B as a member
*/

composition:
    if class A if destroyed, attached class B object should also be destroyed.

Aggregation:
    if class A if destroyed, attached class B object should not be destroyed.
